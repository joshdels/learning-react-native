# Learning by Doing: React Native Mobile Development

This repository contains notes and instructions for setting up and running **React Native** projects using **Expo**.

---

## What is React Native?

React Native is a **framework for building mobile apps** using **JavaScript** and **React**.  
It allows you to write a single codebase that runs on both **Android** and **iOS** devices.

Official documentation: [React Native Docs](https://reactnative.dev/docs/environment-setup)

---

## What is Expo?

**Expo** is a toolchain for React Native that simplifies app development.  
It allows you to:

- Run your app instantly on your device or emulator
- Avoid manual native setup
- Use debugging tools easily

Official tutorial: [Expo Docs](https://docs.expo.dev/tutorial/)

---

## Commands

Start your Expo project:

```bash
npx expo start
```

---

## Android Setup

### 1. Install Android Studio
Download here: [Android Studio](https://developer.android.com/studio)

During installation, make sure to include:

- **Android SDK**  
- **Android SDK Platform**  
- **Android Virtual Device (AVD)**  

for checking

```
adb version
```
